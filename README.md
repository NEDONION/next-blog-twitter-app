
# NextBlogTwitterApp - A Twitter clone App with React and Next.js

## Requisites

- Typescript
- React.js
- Next.js
- TailwindCSS
- Prisma (ORM)
- MongoDB (Database)
- Node.js (Backend)


## Getting Started

```bash
# recommended to use npm
npm install

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
